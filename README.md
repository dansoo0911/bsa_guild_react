# React Приложение

Это React приложение, созданное с помощью Vite.

## Установка

Перед началом работы убедитесь, что у вас установлен Node.js (версия 18 или выше).

Если Node.js не установлен:
1. Скачайте Node.js с официального сайта: https://nodejs.org/
2. Установите его, следуя инструкциям установщика

## Запуск проекта

После установки Node.js выполните следующие команды:

```bash
# Перейдите в папку проекта
cd react_app

# Установите зависимости
npm install

# Запустите сервер разработки
npm run dev
```

Приложение откроется по адресу http://localhost:5173

## Структура проекта

```
react_app/
├── public/          # Статические файлы
├── src/
│   ├── App.jsx      # Главный компонент приложения
│   ├── App.css      # Стили для App
│   ├── main.jsx     # Точка входа приложения
│   └── index.css    # Глобальные стили
├── index.html       # HTML шаблон
├── vite.config.js   # Конфигурация Vite
└── package.json     # Зависимости проекта
```

## Полезные команды

- `npm run dev` - запуск сервера разработки
- `npm run build` - сборка проекта для продакшена
- `npm run preview` - предпросмотр собранного проекта

## Начало работы

Откройте файл `src/App.jsx` и начните редактировать код. Изменения будут автоматически отображаться в браузере благодаря горячей перезагрузке (HMR).
